## Getting Started
[![FOSSA Status](https://app.fossa.com/api/projects/git%2Bgithub.com%2Fhttps-quantumblockchainai-atlassian-net%2Ftoken-gating-template.svg?type=shield)](https://app.fossa.com/projects/git%2Bgithub.com%2Fhttps-quantumblockchainai-atlassian-net%2Ftoken-gating-template?ref=badge_shield)


First, run the development server:

```bash
yarn
yarn next dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
